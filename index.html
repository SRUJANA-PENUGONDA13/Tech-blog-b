Installing and setting up git in local

Created On : 09 Sep 2021

GIT is Source Code Management System. It is free open source version control system to track any kind of projects

This blogs helps to installing and setting up git in local machine

  1. Install git from below official website for windows
    https://git-scm.com/downloads

  2. After git got installed. Open git command prompt in local and tell the git who you are using below commands
    git config --global user.name "YOUR_USERNAME"
    git config --global user.email "YOUR_EMAIL"

  3. After configuration is set use below command to check whether your details are correct or not
    git config --global --list

  4. Setup SSH using by following below steps

    1. Generate SSH keys using below command by using your github email address
      ssh-keygen -t ed25519 -C "github_email_address"

    2. When you're prompted to "Enter a file in which to save the key," press Enter to save in default file location otherwise provide your desired path.
      Enter a file in which to save the key (/c/Users/you/.ssh/id_rsa):[Press enter]

    3. At the prompt, type a secure passphrase. if you have any otherwise press enter
      Enter passphrase (empty for no passphrase): [Type a passphrase]
      Enter same passphrase again: [Type passphrase again]

    4. Add your SSH key to the ssh-agent

      1. Ensure the ssh-agent is running. You can use the "Auto-launching the ssh-agent" instructions in your passphrase, or start it manually using below command
        eval `ssh-agent -s`

      2. Once ssh-agent is started add your ssh-key to your ssh-agent using below command
        ssh-add ~/.ssh/id_rsa

  5. After adding ssh-key to ssh-agent. Copy the SSH public key to clipboard using below command
    clip < ~/.ssh/id_rsa.pub

  6. Copy SSH public key to GITHUB by following below steps

    1. Open your gihub account in the browser

    2. Click on your profile photo in the upper right corner of the page, then click settings

    3. In the user settings sidebar, click SSH and GPG keys.

    4. Click New SSH key or Add SSH key.

    5. In the "Title" field, add a descriptive label for the new key. For example, if you're using a personal Mac, you might call this key "Personal MacBook" and paste your key into "Key" field then save it

  7. Once SSH setup is done, Use below command in your git command prompt to test your ssh connection
    ssh -T git@github.com

  8. Note: If you get any error like Permission denied (publickey) error when using Git, Make sure ssh is running in the same where you saved your ssh-keys initially.
    Use below commands to reregister your SSH key to correct path
    eval `ssh-agent -s`
    ssh-add ~/../You/.ssh/id_rsa

  9. After setting up the configuration and ssh key use below command to initialize your git repository
    git init

  10. Add files to the Staging Area for commit using below command

    1. git add .
    2. git add README.md # To add a specific file
      3.

  11. Before we commit letâ€™s see what files are staged:
    git status # Lists all new or modified files to be committed

  12. Commit Changes you made to your Git Repo:
    git commit -m "First commit"# The message in the " " is given so that the other users can read the message and see what changes you made

  13. Uncommit Changes you just made to your Git Repo:
    git reset HEAD~1 # Remove the most recent commit# Commit again!

  14. Add a remote origin and Push:
    git remote add origin remote_repository_URL

  15. To push changes to origin:
    git push -u origin "branchName"

  16. To show the file changes not yet staged:
    git diff

  17. To clone remote repository:
    git clone remote_repository_URL

  18. To view git commit history:
    git log

  19. To remove all untracked files and folders
    git clean -fd
